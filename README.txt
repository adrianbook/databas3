REQUIREMENTS

Apache Maven 3 or higher

Microsoft SQL Server 2019

DEPLOYMENT

1. Run database-setup/Setup.sql in mssql to set up the database.

2. Open the root directory in a terminal and run the command 'mvn package'.

3. Run the command 'java -jar target/libraryapp.jar' for instructions on how to operate the program.
